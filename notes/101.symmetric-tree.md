# 101. symmetric-tree

## 🔗 题目链接
[LeetCode 101. Symmetric Tree](https://leetcode.com/problems/symmetric-tree/)

## 🧠 题目描述
给你一个二叉树的根节点 `root`，检查它是否是**镜像对称**的（即左右子树互为镜像）。

**示例**：
- 示例 1：输入：root = [1,2,2,3,4,4,3]，输出：true；
  解释：二叉树结构如下，左右子树镜像对称：
  ```
      1
     / \
    2   2
   / \ / \
  3  4 4  3
  ```
- 示例 2：输入：root = [1,2,2,null,3,null,3]，输出：false；
  解释：右子树的3在右侧，左子树的3在左侧，不满足镜像；
- 示例 3：输入：root = []，输出：true（空树视为对称）。

---

## 💡 解题思路
核心思路：
- 递归校验（镜像对比）：判断两棵子树是否镜像，需满足「左子树的左节点 = 右子树的右节点」且「左子树的右节点 = 右子树的左节点」；
- 迭代校验（队列/栈）：用队列/栈存储需要对比的节点对，逐层校验镜像关系，避免递归栈溢出；
- 空节点处理：空节点是镜像对比的边界条件，两个空节点视为对称，一个空一个非空视为不对称。

步骤（以**递归法**为例，最优解法）：
1. 定义递归函数（参数：左子树节点 `left`、右子树节点 `right`）：
   - 终止条件1：`left` 和 `right` 都为空 → 返回 `true`；
   - 终止条件2：`left` 或 `right` 其中一个为空 → 返回 `false`；
   - 终止条件3：`left.val != right.val` → 返回 `false`；
   - 递归校验：左子树的左节点 vs 右子树的右节点，且左子树的右节点 vs 右子树的左节点；
2. 初始调用递归函数：传入根节点的左子树和右子树；
3. 若递归返回 `true`，则整棵树对称，否则不对称。

### 总结
1. 核心逻辑：镜像对称的本质是「左对右、右对左」的节点值相等，递归/迭代都是围绕这一规则校验；
2. 关键细节：空节点的处理是边界核心，必须先判断两个节点是否都为空，再判断值是否相等；
3. 复杂度：
   - 递归法：时间O(n)（遍历所有节点），空间O(n)（递归栈深度）；
   - 迭代法：时间O(n)，空间O(n)（队列存储节点对）。
