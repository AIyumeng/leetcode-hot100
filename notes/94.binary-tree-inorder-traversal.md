# 94. binary-tree-inorder-traversal

## 🔗 题目链接
[LeetCode 94. Binary Tree Inorder Traversal](https://leetcode.com/problems/binary-tree-inorder-traversal/)

## 🧠 题目描述
给定一个二叉树的根节点 `root`，返回它的**中序遍历**结果（左子树 → 根节点 → 右子树）。

**示例**：
- 示例 1：输入：root = [1,null,2,3]，输出：[1,3,2]；
- 示例 2：输入：root = []，输出：[]；
- 示例 3：输入：root = [1]，输出：[1]。

---

## 💡 解题思路
核心思路：
- 递归法：遵循中序遍历的定义（左→根→右），递归遍历左子树，访问根节点，再递归遍历右子树，逻辑简洁直观；
- 迭代法（栈模拟）：用栈模拟递归调用栈，手动控制遍历顺序，避免递归的栈溢出风险；
- 莫里斯遍历（Morris Traversal）：利用二叉树的空指针构建临时链接，实现O(1)空间复杂度的中序遍历（进阶优化）。

步骤（以最常用的**递归法**为例）：
1. 初始化结果列表 `res`，定义递归函数（参数：当前节点 `node`）；
2. 递归函数逻辑：
   - 终止条件：若 `node` 为 `None`，直接返回；
   - 递归左子树：调用递归函数处理 `node.left`；
   - 访问根节点：将 `node.val` 加入结果列表；
   - 递归右子树：调用递归函数处理 `node.right`；
3. 调用递归函数（参数为 `root`），遍历结束后返回结果列表。

### 总结
1. 核心逻辑：中序遍历的核心是「左子树优先」，递归法直接贴合定义，迭代法需用栈模拟递归过程；
2. 复杂度对比：
   - 递归/迭代法：时间复杂度O(n)（遍历所有节点），空间复杂度O(n)（递归栈/显式栈）；
   - 莫里斯遍历：时间复杂度O(n)，空间复杂度O(1)（无额外栈空间）；
3. 应用场景：递归法适合面试快速写代码，迭代法适合处理大数据量避免栈溢出，莫里斯遍历适合对空间要求极高的场景。
