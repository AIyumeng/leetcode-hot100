# 98. validate-binary-search-tree

## 🔗 题目链接
[LeetCode 98. Validate Binary Search Tree](https://leetcode.com/problems/validate-binary-search-tree/)

## 🧠 题目描述
给你一个二叉树的根节点 `root`，判断其是否是一个有效的二叉搜索树（BST）。

**有效二叉搜索树定义**：
1. 节点的左子树只包含**小于**当前节点的数；
2. 节点的右子树只包含**大于**当前节点的数；
3. 所有左子树和右子树自身也必须是二叉搜索树。

**示例**：
- 示例 1：输入：root = [2,1,3]，输出：true；
- 示例 2：输入：root = [5,1,4,null,null,3,6]，输出：false（解释：根节点值为5，右子节点值为4，不满足右子树所有节点大于根节点）；
- 示例 3：输入：root = [2,2,2]，输出：false（BST不允许节点值相等）。

---

## 💡 解题思路
核心思路：
- 递归校验（上下界约束）：为每个节点设置「下界（左子树的最大值）」和「上界（右子树的最小值）」，递归时传递并校验当前节点值是否在合法区间内；
- 中序遍历校验：BST的中序遍历结果是**严格递增**的，通过遍历结果是否递增判断是否为有效BST；
- 避免仅比较子节点：不能只简单比较当前节点与左右子节点值（如右子树的左节点可能小于根节点），必须约束整个子树的范围。

步骤（以**递归上下界约束**为例，最优解法）：
1. 定义递归函数（参数：当前节点 `node`、下界 `lower`、上界 `upper`）：
   - 终止条件：若 `node` 为空，返回 `true`（空树是有效BST）；
   - 校验当前节点：若 `node.val <= lower` 或 `node.val >= upper`，返回 `false`；
   - 递归校验左子树：左子树的上界为当前节点值，下界不变，即 `dfs(node.left, lower, node.val)`；
   - 递归校验右子树：右子树的下界为当前节点值，上界不变，即 `dfs(node.right, node.val, upper)`；
2. 初始调用递归函数：根节点的初始下界为 `-∞`（负无穷），上界为 `+∞`（正无穷）；
3. 若所有递归校验通过，返回 `true`，否则返回 `false`。

### 总结
1. 核心逻辑：递归法通过「上下界约束」精准校验每个节点的合法范围，避免了仅比较子节点的错误；中序遍历法则利用BST「中序递增」的特性，逻辑更直观；
2. 关键细节：BST要求**严格递增**（不能相等），且需约束整个子树的范围（而非仅直接子节点）；
3. 复杂度：
   - 递归法：时间O(n)（遍历所有节点），空间O(n)（递归栈深度）；
   - 中序遍历法：时间O(n)，空间O(n)（栈存储）。
