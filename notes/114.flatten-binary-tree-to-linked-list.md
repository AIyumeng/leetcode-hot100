# 114. flatten-binary-tree-to-linked-list

## 🔗 题目链接
[LeetCode 114. Flatten Binary Tree To Linked List](https://leetcode.com/problems/flatten-binary-tree-to-linked-list/)

## 🧠 题目描述
给你二叉树的根结点 `root`，请你将它展开为一个单链表：
- 展开后的单链表应该同样使用 `TreeNode` 节点，其中 `right` 指针指向链表中下一个节点，而左指针始终为 `null`；
- 展开后的单链表节点顺序应该与二叉树的**先序遍历**顺序相同。

**示例**：
- 示例 1：输入：root = [1,2,5,3,4,null,6]，输出：[1,null,2,null,3,null,4,null,5,null,6]；
  解释：
  原始二叉树结构：
  ```
         1
        / \
       2   5
      / \   \
     3   4   6
  ```
  展开后的链表结构：
  ```
  1 -> 2 -> 3 -> 4 -> 5 -> 6
  （所有左指针为null，仅右指针连接）
  ```
- 示例 2：输入：root = []，输出：[]；
- 示例 3：输入：root = [0]，输出：[0]。

---

## 💡 解题思路
核心思路：
- 递归法（后序遍历）：先递归展开左、右子树，再将左子树移到右子树位置，原右子树接在新右子树的末尾，左指针置空；
- 迭代法（模拟先序遍历）：用栈存储右子节点，先处理左子树，逐步将节点串联为链表；
- 莫里斯遍历（O(1)空间）：利用空右指针构建临时链接，无需栈/递归栈，实现原地展开。

步骤（以递归法为例，最易理解）：
1. 终止条件：若当前节点为空，直接返回；
2. 递归展开左子树：`flatten(root.left)`；
3. 递归展开右子树：`flatten(root.right)`；
4. 暂存原右子树：`temp = root.right`；
5. 调整指针：将左子树移到右子树位置（`root.right = root.left`），左指针置空（`root.left = null`）；
6. 找到新右子树的末尾节点，将暂存的原右子树接在末尾（`cur.right = temp`）；
7. 返回当前节点，完成一层展开。

### 总结
1. 核心逻辑：递归法通过后序遍历「先处理子树、再调整指针」，保证展开顺序符合先序遍历；迭代法/莫里斯遍历则是空间优化版本；
2. 关键细节：必须先将左子树置空，且找到右子树末尾节点后再拼接原右子树，避免指针丢失；
3. 复杂度：
   - 递归法：时间O(n)（遍历所有节点），空间O(n)（递归栈深度，最坏为n）；
   - 迭代法：时间O(n)，空间O(n)（栈存储节点）；
   - 莫里斯遍历：时间O(n)，空间O(1)（原地操作）；
4. 实战建议：面试中优先写递归法，逻辑清晰不易出错；若要求O(1)空间，再使用莫里斯遍历。
