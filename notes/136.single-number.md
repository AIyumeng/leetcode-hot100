# 136. single-number

## 🔗 题目链接
[LeetCode 136. Single Number](https://leetcode.com/problems/single-number/)

## 🧠 题目描述
给定一个**非空**整数数组，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

**说明**：
- 你的算法应该具有线性时间复杂度。 你可以不使用额外空间来实现吗？

**示例**：
- 示例 1：输入：nums = [2,2,1]，输出：1；
- 示例 2：输入：nums = [4,1,2,1,2]，输出：4；
- 示例 3：输入：nums = [1]，输出：1。

---

## 💡 解题思路
核心思路：
- 位运算（异或）：利用异或的三大特性（`a^a=0`、`a^0=a`、异或满足交换律/结合律），将数组所有元素异或，最终结果即为唯一出现一次的数；
- 哈希表统计：遍历数组统计每个数字的出现次数，再遍历哈希表找到次数为1的数字（空间复杂度O(n)，不满足「不使用额外空间」要求）；
- 数学求和：`2*(a+b+c) - (a+a+b+b+c) = c`（即两倍唯一元素和 减 数组总和），但需注意数值溢出风险，且需要额外空间存储唯一元素集合。

步骤（以位运算异或为例，最优解法）：
1. 初始化结果变量 `res = 0`；
2. 遍历数组中的每个数字 `num`：
   - 执行异或操作：`res ^= num`；
3. 遍历结束后，`res` 即为只出现一次的数字，返回该值。

### 总结
1. 核心逻辑：异或解法利用「相同数异或为0，0异或任何数为自身」的特性，是满足「O(n)时间+O(1)空间」的最优解；
2. 关键细节：初始值必须为0（因为0异或第一个数等于该数本身），遍历所有元素无遗漏；
3. 复杂度：
   - 位运算异或：时间O(n)，空间O(1)（仅用常量变量）；
   - 哈希表统计：时间O(n)，空间O(n)；
   - 数学求和：时间O(n)，空间O(n)（存储唯一元素集合）；
4. 实战建议：面试中优先写异或解法，是该题的经典最优解，需熟练掌握异或的核心特性；若忘记位运算，可先写哈希表解法保底。




# 137. single-number-ii

## 🔗 题目链接
[LeetCode 137. Single Number Ii](https://leetcode.com/problems/single-number-ii/)

## 🧠 题目描述
给你一个整数数组 `nums`，除了某个元素只出现**一次**以外，其余每个元素均出现**三次**。请你找出并返回那个只出现了一次的元素。

**说明**：
- 你的算法应该具有线性时间复杂度。你可以不使用额外空间来实现吗？

**示例**：
- 示例 1：输入：nums = [2,2,3,2]，输出：3；
- 示例 2：输入：nums = [0,1,0,1,0,1,99]，输出：99；
- 示例 3：输入：nums = [-2,-2,1,1,4,1,4,4,-4,-2]，输出：-4。

---

## 💡 解题思路
核心思路：
- 位运算（状态机）：统计每个二进制位上1出现的次数，对3取余，最终余数为1的位即为目标数的二进制位（本质是模拟三进制）；
- 哈希表统计：遍历数组统计每个数字的出现次数，再遍历哈希表找到次数为1的数字（空间复杂度O(n)，不满足「不使用额外空间」要求）；
- 数学求和：`3*(a+b+c) - (a+a+a+b+b+b+c) = 2c` → `c = (3*sum(唯一元素集合) - sum(nums)) / 2`（需注意数值溢出，且需要额外空间存储唯一元素）。

步骤（以位运算统计二进制位为例，最优解法）：
1. 初始化结果变量 `res = 0`；
2. 遍历每个二进制位（0到31位，覆盖整数所有位）：
   - 初始化当前位计数 `count = 0`；
   - 遍历数组中的每个数字 `num`：
     - 统计当前位上1的个数：`count += (num >> i) & 1`；
   - 若当前位计数对3取余为1，说明目标数的该位为1：`res |= (1 << i)`；
3. 遍历结束后，`res` 即为只出现一次的数字，返回该值。

### 总结
1. 核心逻辑：位运算解法的关键是「统计每个二进制位的1出现次数，对3取余」，利用整数的二进制特性还原目标数，满足O(n)时间+O(1)空间；
2. 关键细节：需遍历0~31位（包含符号位），确保负数也能正确处理；`(num >> i) & 1` 可精准获取第i位的数值；
3. 复杂度：
   - 位运算统计：时间O(n)（n为数组长度，32次遍历为常量级），空间O(1)；
   - 哈希表统计：时间O(n)，空间O(n)；
   - 数学求和：时间O(n)，空间O(n)（存储唯一元素集合）；
4. 实战建议：面试中优先写二进制位统计解法，逻辑直观且满足空间要求；进阶可掌握「状态机（one/two）」写法（用两个变量模拟三进制状态，效率更高）。
